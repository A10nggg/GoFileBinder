# GoFileBinder   

golang免杀捆绑器 捆绑器免杀效果会持续更新下去-------release处查看新版本，为了更好的免杀之后将不公布源码   
使用以下命令进行捆绑器编译   
`go build GoFileBinder.go`
   
使用以下命令生成捆绑文件(注：GoFileBinder.exe单独放入一个文件夹中)   
`GoFileBinder.exe	木马.exe xxx.txt`
